## Installment

```console
$ npm install
```

Create `.env` file and create these variables:

```
PORT
COOKIE_SECRET
GMAIL_NAME
GMAIL_USER
GMAIL_PASS
```

Also create `pulse.json` file (at `config/db/pulse.json`) and create these variables:

```json
{
    "host": "<hostname>",
    "user": "<user>",
    "password": "<password>",
    "database": "<database>",
    "multipleStatements": true
}
```
